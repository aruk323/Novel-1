# ROUTE_DESIGN.md
# 『愛のアブダクション』分岐設計資料

## 1. 本資料の目的

本資料は、`docs/STORY_BIBLE.md` と `docs/CHAPTER_PLAN.md` に書かれている構想を優先し、第5章終了時から第6章A〜Eへ分岐し、全9個のエンディングへ到達するための分岐設計を固定するための資料である。

重要な固定事項：

- 第6章のルート数は、A、B、C、D、Eの5ルートとする。
- エンディング数は合計9個とする。
- 各ルートから到達するエンディング数は以下で固定する。
  - 第6章A：1個
  - 第6章B：2個
  - 第6章C：3個
  - 第6章D：1個
  - 第6章E：2個
- ルート数やエンディング数を実装都合で増減させない。

## 3. 各パラメータの意味

### 3.1 `affection`

リィナと主人公の関係が、観察者とサンプルという非対称な関係から、互いを一人の存在として認める関係へ近づいた度合いを表す。

高いほど、リィナは主人公を単なる地球個体ではなく、失いたくない相手として扱う。

### 3.2 `invasion`

地球侵略計画への肯定、母星側の合理性への同調、地球を捨ててもリィナを選ぶ姿勢を表す。

高いほど、第6章C「地球制圧作戦」や、第6章B「銀緑の同族」へ入りやすくなる。

### 3.3 `dependency`

リィナの主人公への執着、独占欲、不安、隔離欲求を表す。

高いほど、第6章D「君だけが地球」へ入りやすくなる。ヤンデレ展開は急に発生させず、観察、保護、接触継続、不安、通信制限、隔離の順で段階的に強める。

### 3.4 `assimilation`

主人公の肉体・精神がリィナの種族へ近づいている度合いを表す。

高いほど、第6章B「銀緑の同族」へ入りやすい。ただし、洗脳を伴う同族化と、自由意志を保った同族化は明確に分ける。

### 3.5 `wariness`

主人公がリィナや母星文明を危険視している度合いを表す。

高い場合、洗脳拒否、地球防衛、別離の選択に説得力を持たせる補助値として使う。

### 3.6 `earth_empathy`

リィナが地球文化、地球人の感情、非効率な日常の価値を理解した度合いを表す。

高い場合、第6章A「侵略者の反逆」や第6章E「彼女が地球を選ぶ日」に説得力を持たせる。

### 3.7 `free_will_respect`

主人公がリィナを命令に縛られた侵略者としてだけではなく、自分の意思で未来を選べる存在として扱った度合いを表す。

高い場合、第6章A「侵略者の反逆」へ入りやすい。

## 4. 第1章〜第5章での加算方針

数値の具体的なしきい値や加算値は、実装時に調整する。ただし、どの章でどのパラメータを主に動かすかは以下で固定する。

### 4.1 第1章：銀緑の来訪者

主に加算・記録するもの：

- `affection`
- `wariness`
- `earth_empathy` の前提となる地球文化への興味

加算方針：

- リィナに名前で呼ぶよう求める、会話を続ける、彼女を単なる敵として扱わない選択で `affection` を上げる。
- 脱出未遂、UFO内ルールへの反発、侵略目的への警戒で `wariness` を上げる。
- 食事や地球の生活を説明する選択で、後続章の `earth_empathy` 加算の前提を作る。

### 4.2 第2章：UFO内共同生活実験

主に加算するもの：

- `affection`
- `dependency`
- `wariness`

加算方針：

- 共同生活、手を繋ぐ実験、重力制御トラブルでリィナを受け入れる選択は `affection` を上げる。
- リィナの観察癖や接触継続を許容する選択は `dependency` を上げる。
- 催眠、記憶操作、同族化などアンテナ能力を危険視する選択は `wariness` を上げる。

### 4.3 第3章：地球文化調査と偽装デート

主に加算するもの：

- `affection`
- `earth_empathy`
- `invasion`
- `has_earthling_path_flag`

加算方針：

- コンビニ、学校、ゲームセンター、公園などで地球文化の楽しさを伝える選択は `earth_empathy` を上げる。
- 偽装デートでリィナの感情変化を受け止める選択は `affection` を上げる。
- 地球文化を侵略や支配に利用する方向へ誘導する選択は `invasion` を上げる。
- リィナが地球で生きる可能性を示す選択は `has_earthling_path_flag` を立てる候補にする。

### 4.4 第4章：侵略計画と恋愛感情のバグ

主に加算するもの：

- `dependency`
- `invasion`
- `affection`
- `free_will_respect`

加算方針：

- リィナの不安、嫉妬、喪失恐怖を受け入れすぎる選択は `dependency` を上げる。
- 侵略計画に理解や協力を示す選択は `invasion` を上げる。
- リィナの感情をバグではなく大切なものとして扱う選択は `affection` と `free_will_respect` を上げる。
- リィナと距離を取る選択は `wariness` を補助的に上げてもよい。

### 4.5 第5章：同族化実験と心の暴走

主に加算・確定するもの：

- `assimilation`
- `has_alienization_flag`
- `refused_mind_control`
- `accepted_mind_control`
- `has_earthling_path_flag`
- `respected_riina_free_will`
- 第6章A〜Eの分岐に関わる最終補正

重要選択肢ごとの扱い：

| 選択肢 | 主な影響 |
|---|---|
| リィナを信じて装置に入る | `affection` と `assimilation` を上げ、`has_alienization_flag` を立てる候補にする。 |
| 絶対に拒否する | `refused_mind_control` を立てる候補にし、`wariness` を上げる。 |
| 洗脳なしなら受け入れる | `assimilation` を上げ、`has_alienization_flag` と `refused_mind_control` を立てる候補にする。 |
| リィナも地球人になればいいと言う | `has_earthling_path_flag` を立て、`earth_empathy` と `free_will_respect` を上げる。 |

第5章の最後では、リィナの制御コード暴走、アンテナビーム暴走、主人公の呼びかけ、侵略艦隊出現を経て、第6章A〜Eのいずれかへ必ず分岐させる。

## 5. 第5章終了時の第6章A〜Eへの分岐条件

以下の条件は、物語構造を固定するための設計条件である。具体的なしきい値は実装時に定義する。

### 5.1 第6章A：侵略者の反逆

条件：

- `affection` が50以上
- `invasion` が0
- `free_will_respect` が20以上、または `respected_riina_free_will` が立っている。
- リィナを母星の命令ではなく、自分の意思で未来を選ぶ存在として扱っている。

### 5.2 第6章B：銀緑の同族

条件：

- `affection` が30以上。
- `invasion` が2。
- `assimilation` が3以上。
- `has_alienization_flag` が立っている。

### 5.3 第6章C：地球制圧作戦

条件：

- `invasion` が2。
- または、第6章A、B、D、Eのいずれにも当てはまらない。

第6章Cは、侵略度が最大の場合のルートであると同時に、フォールバック先でもある。

### 5.4 第6章D：君だけが地球

条件：

- `affection` が40以上。
- `dependency` が5。

### 5.5 第6章E：彼女が地球を選ぶ日

条件：

- `affection` が30以上。
- `invasion` が1。
- `dependency` が2〜4。
- `earth_empathy` が5以上、または `has_earthling_path_flag` が立っている。

## 6. 第6章A〜Eそれぞれの目的

### 6.1 第6章A：侵略者の反逆

リィナが母星に反逆し、地球を守る側に立つルート。

山場では、リィナが上官に「私は地球を侵略しない」と宣言する。主人公とリィナは、母星艦隊の中枢へアクセスするためにUFOを突入させる。

### 6.2 第6章B：銀緑の同族

主人公の宇宙人化が進み、リィナとの精神リンクが強まるルート。

山場では、主人公が洗脳を受け入れるか、自分の意思を保つかを選ぶ。

### 6.3 第6章C：地球制圧作戦

地球侵略が進行するルート。

山場では、リィナが母星に反発するか、侵略を進めるかで悩む。侵略、別離、敗北の3方向へ分岐する。

### 6.4 第6章D：君だけが地球

リィナの愛情が依存と独占に傾いていくルート。

山場では、リィナが地球側の人間関係を「不要な接続」と判断する。リィナは母星も地球も捨て、主人公だけを閉じた世界に置こうとする。

### 6.5 第6章E：彼女が地球を選ぶ日

リィナが宇宙人としての能力を捨て、地球人に近い存在になる可能性を探るルート。

山場では、リィナがアンテナ能力を封印するかどうかを選ぶ。

## 9. 条件が競合した場合の優先順位

条件競合時は、以下の順で第6章ルートを決定する。

1. 第6章D：君だけが地球
2. 第6章A：侵略者の反逆
3. 第6章B：銀緑の同族
4. 第6章E：彼女が地球を選ぶ日
5. 第6章C：地球制圧作戦

### 9.1 第6章Dを最優先する理由

依存度が高く、リィナの愛情が独占と隔離へ傾いている場合は、ヤンデレエンドへの積み重ねを無効化しないため、第6章Dを優先する。

### 9.2 第6章Aを第6章Bより優先する理由

第6章Aは、好感度が非常に高く、侵略度が低く、リィナの自由意志を尊重している場合の真ハッピーエンド直行ルートである。

同族化よりも、リィナ自身が母星の命令から自由になる選択を優先する。

### 9.3 第6章Bを第6章Eより優先する理由

第6章Bは、宇宙人化フラグが立っている場合の専用ルートである。主人公の同族化が進んでいる場合は、地球人化ルートよりも同族化ルートを優先する。

### 9.4 第6章Eを第6章Cより優先する理由

第6章Eは、好感度が高く、侵略度と依存度が中程度で、リィナが地球を選ぶ可能性を持つルートである。

第6章A、B、Dほど極端な条件ではないが、地球人化フラグや地球理解が積み上がっている場合は、侵略ルートへ落とす前に第6章Eを判定する。

## 10. フォールバック先

どの条件にも当てはまらない場合は、第6章C「地球制圧作戦」へ進む。

第6章Cは、侵略度が最も高い場合のルートであると同時に、他のルートに当てはまらない場合の自動移行先として扱う。

## 11. 実装時の注意点

### 11.1 ルート数を増やさない

第6章はA、B、C、D、Eの5ルートで固定する。

STORY_BIBLE上の「主人公が地球を守るためにリィナと別れるルート」は、第6章C内の地球防衛・別離エンドとして扱う。独立した第6章Fを作らない。

### 11.2 エンディング数を増やさない

エンディングは9個で固定する。

小さな差分演出や後日談差分が必要な場合でも、エンディングIDは増やさず、同一エンディング内の演出差分として扱う。

### 11.3 洗脳あり同族化と洗脳なし同族化を混同しない

第6章Bでは、主人公の自由意志が失われるかどうかが最重要分岐になる。

- 洗脳あり：主人公宇宙人化＋洗脳エンド「あなたも侵略者」
- 洗脳なし：共犯宇宙カップルエンド「一緒に宇宙へ」

### 11.4 リィナを単純な悪役にしない

侵略ルートや敗北エンドでも、リィナをただの冷酷な敵にしない。

母星の命令、感情抑制コード、未熟さ、主人公への感情が絡み合った結果として描く。

### 11.5 ヤンデレを急に発生させない

第6章Dは、依存度の積み重ねがある場合のみ成立させる。

序盤から突然監禁へ飛ばさず、観察、保護、接触継続、不安、通信制限、隔離という段階を踏む。

### 11.6 地球人化エンドでリィナの個性を消さない

第6章Eから地球人化エンドに進む場合でも、リィナを完全に普通の少女にしない。

無表情気味、理屈っぽさ、ズレた言動、銀髪や緑の瞳などの個性は残す。

### 11.7 第6章突入後に別ルートへ横滑りさせない

第6章に入った後は、基本的にそのルート内のエンディング条件だけを調整する。

- Aは1エンド直行。
- Bは2エンド内で分岐。
- Cは3エンド内で分岐。
- Dは1エンド直行。
- Eは2エンド内で分岐。

### 11.8 最終検証項目

実装後は、最低限以下を検証する。

- 第6章Aへ到達できる。
- 第6章Bへ到達できる。
- 第6章Cへ到達できる。
- 第6章Dへ到達できる。
- 第6章Eへ到達できる。
- 第6章Aから到達するエンディングは1個。
- 第6章Bから到達するエンディングは2個。
- 第6章Cから到達するエンディングは3個。
- 第6章Dから到達するエンディングは1個。
- 第6章Eから到達するエンディングは2個。
- 合計エンディング数は9個。
- どの条件にも当てはまらない場合、第6章Cへ進む。

### 12.6 バランス方針

- 1つの選択肢で変化する値は原則 `+1`、強い関係進行や第5章の重要選択肢のみ `+2` とする。

- 第5章終了時に第6章A〜Eのいずれにも該当しない場合は、第6章Cへフォールバックする。
