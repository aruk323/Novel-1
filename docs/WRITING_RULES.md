# WRITING_RULES.md
# 『愛のアブダクション』執筆ルール

## 1. 基本文体

本作は、スマホ向け長編ノベルゲームとして読みやすい文体にする。

地の文は長すぎず、タップで読み進めやすいテンポを重視する。1メッセージあたりの文章量は、スマホ画面で読み切れる長さに抑える。

ラブコメ部分では軽快に、不穏なSF部分では静かに冷たく、終盤では感情の重みを出す。

---

## 2. 地の文の雰囲気

### 2.1 序盤

非日常感と戸惑いを重視する。UFOやリィナの描写は、主人公が未知のものを見ている感覚を大切にする。

例：

銀色の壁には、血管のように緑色の光が走っていた。  
病院にも、研究室にも似ている。  
けれど、そのどちらよりも冷たく、どちらよりも現実離れしていた。

### 2.2 ラブコメ場面

主人公のツッコミとリィナの真顔のズレを活かす。地の文はテンポよく、余計な説明を入れすぎない。

例：

リィナは真剣だった。  
本気で、プリンを地球文明の中枢物資だと思っている顔だった。  
無表情なので分かりにくいが、アンテナの先がいつもより明るい。

### 2.3 不穏な場面

音、光、沈黙を使う。過度に叫ばせない。リィナは静かだからこそ怖い。

例：

アンテナの先端が、淡い緑から鋭い光へ変わった。  
リィナの表情は変わらない。  
だからこそ、その光だけが、彼女の中で何かが壊れかけていることを示していた。

### 2.4 終盤

派手な戦闘描写より、主人公とリィナの選択を中心にする。宇宙艦隊や地球規模の危機は背景として描き、二人の会話を最重要にする。

---

## 3. 会話の長さ

### 3.1 基本

1つの台詞は短めにする。スマホ画面で読みやすいよう、長台詞を避ける。

ただし、感情告白、世界観説明、ルート分岐前の重要会話では、やや長い台詞を使ってよい。

### 3.2 リィナの台詞

リィナの台詞は、短く、淡々と、観察的にする。

良い例：

「心拍数上昇。照れている？」

「否定は不要。データは正直」

「君の笑顔を再要求する」

悪い例：

「もう、そんなこと言われたら恥ずかしくてどうしたらいいかわからないよ！」

このような普通の感情表現はリィナらしくない。感情が高まった場面でも、言葉は不器用にする。

### 3.3 主人公の台詞

主人公は常識的なツッコミ役。プレイヤーが感情移入しやすいように、過度に個性を固定しすぎない。

良い例：

「それ、たぶんデートじゃなくて実験だよね」

「プリンを侵略兵器みたいに扱わないで」

「リィナは、本当にそれを自分で選びたいの？」

### 3.4 ネムの台詞

船内AIは丁寧な機械口調。時々リィナの感情を暴露してコメディにする。

良い例：

「リィナ様の心拍数上昇を確認。照れの可能性が82%です」

「記録名『ユーザー_笑顔_保存用』は、研究資料として不自然です」

## 4. 文体ルール

### 4.1 一文を長くしすぎない

スマホ向けなので、長い複文を避ける。重要な文章は短く区切る。

悪い例：

リィナは主人公のことを観察しているつもりだったが、実際には観察対象として以上に興味を持っていて、そのことを本人は認めようとせず、胸の違和感を船内気圧の問題だと説明した。

良い例：

リィナは主人公を観察している。  
本人はそう言い張った。  
けれど、観察記録の量は明らかに異常だった。  
胸の違和感については、船内気圧の問題だと説明していた。

### 4.2 難しいSF用語を連発しない

SF設定は雰囲気として使う。専門用語を並べすぎず、リィナの行動や事件を通して理解させる。

### 4.3 ラブコメと不穏さを交互に置く

甘い場面だけが続くと侵略設定が薄くなる。怖い場面だけが続くとラブコメとして重すぎる。

基本リズム：

1. リィナのズレた行動で笑わせる
2. 主人公との距離が少し縮まる
3. リィナの危険性や母星の命令を思い出させる
4. それでも主人公が彼女を気にしてしまう

### 4.4 リィナの感情表現は直接ではなく間接で出す

リィナは感情を言葉で説明するのが苦手。表情よりも、以下で表現する。

- アンテナの揺れ
- 発光ラインの明滅
- 視線の逸れ
- 記録名の異常
- 船内AIの指摘
- 理屈っぽい言い訳
- 主人公への接触時間の増加

## 5. 選択肢の作り方

## 5.1 選択肢は3つ前後を基本にする

スマホ画面で押しやすいように、通常選択肢は3つ。重要分岐では4〜5つまで許可する。

### 通常選択肢の例

- 正直に感想を言う
- リィナをからかう
- 黙って受け取る

### 重要選択肢の例

- リィナは命令の道具じゃない
- 一緒に侵略しよう
- 自分をリィナと同じにして
- 地球を選ぶ。リィナを止める
- どこにも行かないで

## 5.2 各選択肢は感情の方向性を持たせる

選択肢は単なる正解・不正解にしない。どの選択にも魅力とリスクを持たせる。

例：

「リィナを信じて装置に入る」  
好感度は上がるが、同族化度も上がる。

「絶対に拒否する」  
自由意志は守れるが、リィナとの距離が開く可能性がある。

「洗脳なしなら受け入れる」  
真エンド条件になるが、好感度が低いと失敗する。

## 5.3 パラメータ変化を意識する

主なパラメータ：

- 好感度
- 侵略度
- 同族化度
- 依存度
- 地球人化フラグ
- 洗脳拒否フラグ

選択肢ごとに、どの値が上がるかを内部的に設定する。

## 5.4 選択肢直後のリアクションを必ず入れる

選択肢を選んだら、リィナや主人公の反応を短く入れる。パラメータだけ変えて次へ進まない。

例：

主人公が「もう少しだけ手を繋いでいる」を選んだ場合：

リィナは手を離さなかった。  
むしろ、指先に少しだけ力が入る。  
「安全確認を継続する。……必要な処置」

## 6. キャラ崩れ防止ルール

### 6.1 リィナは感情をすぐ認めない

「好き」「寂しい」「嫉妬した」と直接言うのは、物語後半または重要場面だけ。序盤から中盤は、必ず別の言葉に置き換える。

置き換え例：

- 好き → 観察優先度が高い
- 寂しい → 会話不足による演算低下
- 嫉妬 → 未知の不快反応
- 会いたい → 追加観察が必要
- 手を繋ぎたい → 接触実験を再実施する

### 6.2 リィナは急にテンション高くならない

無表情クール系を維持する。嬉しい時も、声のトーンやアンテナで表現する。

悪い例：

「やったー！ プリン大好き！」

良い例：

「プリンを追加で要求する。これは、文化理解のため」

アンテナの先が、さっきより明るく光っていた。

### 6.3 主人公は倫理観を完全に消さない

侵略ルートや共犯ルートでも、主人公が地球を裏切る重みは描く。軽いノリだけで地球侵略を肯定しない。

### 6.4 洗脳エンドは不穏にする

洗脳エンドを甘いだけにしない。リィナの隣にいられる幸福感と、主人公の自由意志が薄れている怖さを両立する。

### 6.5 地球侵略エンドは平和すぎる理想郷にしない

リィナの支配によって争いが減ったとしても、自由が失われていることを示す。

### 6.6 真エンドはご都合主義にしない

真エンドでは、リィナが簡単に母星を完全撃破するのではなく、地球を保護観察惑星として扱うなど、現実的な落としどころを作る。

### 6.7 地球人化エンドでリィナの能力喪失を軽く扱わない

アンテナ能力の封印は、リィナにとって大きな喪失である。地球で生きる喜びと同時に、宇宙人としての自分を手放す痛みも描く。

### 6.8 ヤンデレエンドは主人公の選択の結果にする

リィナが突然狂うのではなく、主人公が依存を受け入れ続けた結果として発生させる。

## 7. シーン作成テンプレート

新しいシーンを書く時は、以下の形を使う。

```text
## シーン名

### 目的
このシーンで何を進めるか。

### 舞台
どこで起きるか。

### 登場キャラクター
誰が出るか。

### 感情の開始点
シーン冒頭で、主人公とリィナはどんな感情か。

### 出来事
シーン内で起きること。

### 選択肢
必要なら選択肢を置く。

### パラメータ変化
好感度、侵略度、同族化度、依存度など。

### 感情の終了点
シーン終了時、二人の関係がどう変化したか。

### 次シーンへの引き
次に読み進めたくなる一文や事件。
```

## 8. 台詞作成テンプレート

### リィナの通常台詞

```text
「観察結果を更新。君は、予想よりも騒がしい」
```

### リィナの照れ隠し台詞

```text
「発光ラインの明滅は、船内温度の影響。私の感情とは無関係」
```

### リィナの危険台詞

```text
「抵抗しないで。記憶編集は、痛みを最小限に抑える」
```

### リィナの告白に近い台詞

```text
「君を失いたくない。これは命令ではない。たぶん、私の意思」
```

### 主人公のツッコミ台詞

```text
「それ、地球ではデートっていうんだよ。侵略準備じゃなくて」
```

### 主人公の真剣台詞

```text
「リィナは、命令じゃなくて、自分で選んでいいんだ」
```

## 9. 章本文を書く時の優先順位

1. 主人公とリィナの関係が変化しているか
2. リィナの無表情クールさとポンコツさが両立しているか
3. 侵略者としての危険性が残っているか
4. 選択肢が後の分岐に意味を持っているか
5. スマホで読みやすい文量になっているか
6. ラブコメとSF不穏要素のバランスが取れているか
7. 章ごとのラストに次を読みたくなる引きがあるか

---

## 10. 禁止表現・避けるべき書き方

### 10.1 説明だけで進むシーン

長い世界観説明だけのシーンは避ける。必ず主人公とリィナの会話、行動、感情変化を入れる。

### 10.2 リィナのキャラ崩れ

リィナが急に普通の明るい少女のように振る舞うのは禁止。感情を得ても、無表情、不器用、観察口調は維持する。

### 10.3 主人公の無意味な鈍感化

ラブコメだからといって、主人公を極端に鈍感にしない。リィナの変化に気づく場面を入れる。

### 10.4 過度に暗いだけの展開

本作はラブコメSFである。重い展開があっても、リィナのズレた言動や主人公のツッコミで呼吸できる場面を作る。

### 10.5 選択肢の結果が見えない

選択肢を選んだ後、何が変わったのか読者が感じられない展開は避ける。小さな反応でよいので、必ず変化を描く。
